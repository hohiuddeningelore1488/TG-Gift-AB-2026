## How to Install / Use

### Quick Start

1.  **Clone Repository**: `git clone https://github.com/yourusername/telegram-gift-automation-project-toolkit-2026.git`
2.  **Navigate Directory**: `cd telegram-gift-automation-project-toolkit-2026`
3.  **Install Dependencies**: `pip install -r requirements.txt`
4.  **Configure Settings**: Edit `config.yaml` with your Telegram API credentials and purchase details.
5.  **Run Automation**: `python main.py --purchase --gift-type premium --account user1`
6.  **Monitor Logs**: Review the `logs/automation.log` file for transaction status.
